## 安装与系统要求

**适用场景**：第一次在电脑上安装知聊，或从旧版本升级。

| 项 | 要求 |
|----|------|
| 系统 | Windows 10/11 x64（当前安装包） |
| 微信 | PC 微信 **4.0 及以上** |
| 磁盘 | 建议预留 500MB+（含导出与缓存） |
| 权限 | 若无法读取微信进程，可 **右键 ZhiLiao → 以管理员身份运行** |

**安装步骤**

1. 下载 `ZhiLiao-5.2.0-Setup.exe`（见仓库 [releases](../releases/) 或 GitHub Releases）。
2. 运行安装向导，按提示完成安装。
3. 从开始菜单或桌面打开 **知聊**（实际启动 `ZhiLiao.exe` → `WeFlow.exe`）。

---

## 首次启动与连接数据库

1. 启动知聊，在引导页阅读协议并同意。
2. 在 **数据库路径** 处点击 **自动检测**；失败时手动选择微信数据根目录（Windows 一般为 `文档\xwechat_files`）。
3. 选择当前要分析的 **微信账号（wxid）**；多账号可在 **账号管理** 中切换。
4. 按提示完成 **数据库密钥** 获取；连接成功后侧边栏各功能可用。
5. 若仅能看到首页 / 设置而无法进入聊天，说明尚未连库成功，请返回引导页重试。

**数据目录**：`文档\WeFlow\`（配置、导出、定时任务等均在此）

## 界面说明

**左侧边栏**（自上而下）：

| 入口 | 说明 |
|------|------|
| **首页** | 功能入口（聊天整理 / SOP、AI 配置、导出中心） |
| **聊天** | 浏览、搜索本地聊天记录 |
| **朋友圈** | 朋友圈时间线与媒体 |
| **通讯录** | 好友、群聊、公众号 |
| **分析** | 私聊 / 群聊统计 |
| **年度报告** | 年度可视化报告 |
| **导出** | 导出中心（HTML、Markdown 等） |
| **设置** | 账号、AI、Agent 连接、备份等 |

**授权**：首次使用需完成 **授权管理**（设置 → 账号）；未授权时仅可进入设置修改激活码。

## 主要功能

### 1. 查看与搜索聊天

**聊天** 页选择会话，可滚动浏览历史消息，支持搜索与防撤回查看（若已开启）。

### 2. 聊天整理 / SOP（特供）

**首页** 或 `/sop` 进入 **聊天整理工作台**：选择群聊与时间范围，生成 Word / Markdown / JSON 归档。

### 3. 导出中心

**导出** 页支持批量导出聊天记录；可配置 **定时导出**（设置 → 定时导出，或 `Documents/WeFlow` 下配置文件 + 计划任务）。

### 4. 分析与报告

- **分析**：私聊 / 群聊画像与统计  
- **年度报告 / 双人报告**：可视化长报告  

### 5. 朋友圈

浏览本地解密后的朋友圈图片、视频与实况。

### 6. 备份与应用锁

**设置** 中可配置数据备份、Windows Hello / 应用锁等。

## 本地 HTTP API（供知销等调用）

若需 **知销** 读取微信个人聊天，须在本机开启 API：

1. 知聊 → **设置 → Agent 连接**（或「本地 HTTP API」相关项）。
2. 启用 API，记下 **端口**（默认 `5031`）与 **Token**。
3. 保持知聊运行；可用浏览器或 curl 验证：

```http
GET http://127.0.0.1:5031/api/v1/status
Authorization: Bearer <你的Token>
```

4. 在 **知销 → 设置 → 微信数据** 中填写相同地址与 Token。

详细 API 说明见源码仓库 `docs/features/http-api.md`（开发者参考）。

## 常见问题

| 现象 | 处理 |
|------|------|
| 连库失败 / 密钥错误 | 确认微信 4.0+、路径正确；Windows 可尝试管理员运行 |
| 任务管理器显示 WeFlow.exe | 正常；知聊外壳为 ZhiLiao.exe，WCDB 必须在 WeFlow 进程内 |
| 知销连不上 API | 确认知聊已启动、API 已启用、Token 一致、防火墙未拦截 127.0.0.1 |
| macOS / Linux | 当前本仓库安装包为 Windows；其他平台见源码仓库自行打包 |
